# HR Attrition Dashboard

## Project Overview

This project presents an interactive HR Attrition Dashboard developed using Microsoft Power BI. The dashboard helps analyze employee attrition trends and identify factors affecting employee retention.

## Tools Used

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Microsoft Excel

## Key Features

* Employee Attrition Analysis
* Department-wise Attrition Insights
* Age Group Analysis
* Job Role Analysis
* Salary Range Analysis
* Overtime Impact Analysis
* Gender Analysis
* Education Field Analysis
* Job Satisfaction Analysis
* Interactive Department Slicers

## Key Performance Indicators (KPIs)

* Total Employees
* Total Attrition
* Average Salary
* Attrition Rate

## Dashboard Pages

### 1. HR Overview Dashboard

Provides high-level attrition insights based on:

* Department
* Age Group
* Job Role
* Salary Range
* Overtime

### 2. Employee Insights Dashboard

Provides detailed employee analysis based on:

* Gender
* Education Field
* Marital Status
* Job Satisfaction

## Key Findings

* Employees working overtime show higher attrition.
* Younger employees contribute significantly to attrition.
* Sales and Research & Development departments show higher employee turnover.
* Salary and job satisfaction influence employee retention.

## Project Outcome

The dashboard transforms raw HR data into actionable insights, helping organizations understand employee attrition patterns and improve workforce planning.
